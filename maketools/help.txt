Usage: make [-jN] TARGET ARGS

    help                Print this help.
    make-help           Print targets related to development of the make-system
    readme              Print README.

  Build:
    clean               Clean build and intermediate artifacts
    package             Build packages

  Development:
    check               Run static, test, scan.
    format              Format source code.
    scan                Run vulnerability scan.
    static              Run static code analysis.
    systest             Run system test.
    test                Run all unit test.

  Tools:
    poetry              Installs poetry

Set Q= , to echo commands run in recipes.
